# 过程能力指数 (Process Capability, Cp/Cpk)

## 定义

过程能力指数是衡量生产过程满足产品规格要求能力的统计指标。

## 计算公式

\[
Cp = \frac{USL - LSL}{6\sigma}
\]

\[
Cpk = \min\left(\frac{USL - \mu}{3\sigma}, \frac{\mu - LSL}{3\sigma}\right)
\]

其中：
- USL = 上规格限
- LSL = 下规格限
- μ = 过程均值
- σ = 过程标准差

## 目标值

Cpk ≥ 1.33

## 数据来源

- MES 系统
- 统计过程控制系统

## 更新频率

每周
