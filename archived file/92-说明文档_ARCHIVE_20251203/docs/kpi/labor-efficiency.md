# 人工效率 (Labor Efficiency)

## 定义

人工效率是产出工时与员工实际出勤时间的比值，衡量人力资源利用效率。

## 计算公式

\[
\text{Labor Efficiency}(\%) = \frac{\text{产出工时}}{\text{实际出勤时间}} \times 100\%
\]

其中：
- **产出工时** = 实际产量 × 标准工时
- **实际出勤时间** = 员工出勤小时数

## 目标值

≥ 90%

## 数据来源

- MES 系统
- 考勤系统

## 更新频率

每周
