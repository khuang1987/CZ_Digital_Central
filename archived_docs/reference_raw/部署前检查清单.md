## ⚠️ 可能遇到的问题

### 问题 1：网络连接失败

**症状：**
```
fatal: unable to access 'https://github.com/...': Recv failure: Connection was reset
```

**解决方法：**
1. 检查网络连接
2. 如果在公司网络，可能需要配置代理
3. 或使用 SSH 方式

---

### 问题 2：权限错误

**症状：**
```
Permission denied (publickey)
```

**解决方法：**
1. 配置 SSH 密钥
2. 或使用 Personal Access Token

---

### 问题 3：构建警告

**症状：**
```
WARNING - Doc file '...' contains a link '...', but the target is not found
```

**说明：**
- 这些是警告，不会阻止部署
- 主要是一些锚点链接问题
- 可以后续修复
