# XLSX转CSV工具

这是一个简单易用的图形界面工具，用于将Excel文件（.xlsx/.xls）转换为CSV格式文件。

## 功能特点

- 🖱️ 图形化界面，操作简单
- 📁 支持选择输入文件和输出目录
- 📊 支持多工作表Excel文件转换
- 🔧 可自定义编码格式和分隔符
- 📈 实时显示转换进度
- 📝 详细的转换日志
- ⚡ 多线程处理，界面不卡顿

## 安装要求

### Python版本
- Python 3.7 或更高版本

### 依赖包
```bash
pip install -r requirements.txt
```

或者手动安装：
```bash
pip install pandas openpyxl xlrd
```

## 使用方法

1. **运行程序**
   ```bash
   python xlsx_to_csv_converter.py
   ```

2. **选择文件**
   - 点击"浏览"按钮选择要转换的Excel文件
   - 支持.xlsx和.xls格式

3. **设置输出目录**
   - 点击"选择目录"按钮选择CSV文件保存位置
   - 默认保存在原文件所在目录

4. **配置转换选项**
   - **编码格式**: 选择CSV文件的编码格式
     - `utf-8-sig`: 推荐，支持中文且Excel可正确识别
     - `utf-8`: 标准UTF-8编码
     - `gbk`: 中文GBK编码
     - `gb2312`: 中文GB2312编码
   
   - **分隔符**: 选择CSV文件的分隔符
     - `,`: 逗号分隔（标准CSV）
     - `;`: 分号分隔（欧洲标准）
     - `\t`: 制表符分隔（TSV格式）

5. **开始转换**
   - 点击"开始转换"按钮
   - 程序会显示转换进度和日志信息
   - 转换完成后会弹出成功提示

## 输出文件命名规则

- **单工作表文件**: `原文件名.csv`
- **多工作表文件**: `原文件名_工作表名.csv`

例如：
- `data.xlsx` → `data.csv`
- `report.xlsx` (包含Sheet1, Sheet2) → `report_Sheet1.csv`, `report_Sheet2.csv`

## 注意事项

1. **文件格式支持**
   - 支持.xlsx格式（推荐）
   - 支持.xls格式（旧版Excel）

2. **数据完整性**
   - 转换过程中会保留所有数据
   - 空单元格会转换为空字符串
   - 不会丢失格式信息（仅保留数据）

3. **中文支持**
   - 推荐使用`utf-8-sig`编码以确保Excel正确显示中文
   - 如果Excel显示乱码，可尝试`gbk`编码

4. **大文件处理**
   - 程序支持处理大型Excel文件
   - 转换过程在后台线程执行，不会阻塞界面

## 常见问题

**Q: 转换后的CSV文件在Excel中显示乱码？**
A: 请尝试将编码格式改为`utf-8-sig`或`gbk`。

**Q: 如何处理包含特殊字符的数据？**
A: 程序会自动处理特殊字符，如果仍有问题，请检查原始Excel文件的编码。

**Q: 转换速度很慢？**
A: 大文件转换需要时间，请耐心等待。程序会显示实时进度。

**Q: 程序无法启动？**
A: 请确保已安装所有依赖包：`pip install pandas openpyxl xlrd`
