# 报表说明总览

## 报表分类

本节提供数据平台所有报表的详细说明和使用指南，帮助用户快速理解和使用各类报表。

### 按部门分类
- **供应链部门报表**：计划达成率、库存周转、供应商绩效等
- **生产部门报表**：生产效率、设备利用率、良率分析等  
- **质量部门报表**：质量合格率、不良品分析、客诉统计等

### 按使用场景分类
- **管理层报表**：高层决策支持、关键指标监控
- **运营报表**：日常业务跟踪、异常预警
- **分析报表**：深度数据分析、趋势预测
