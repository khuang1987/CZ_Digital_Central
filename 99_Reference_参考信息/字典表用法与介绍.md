# 字典表用法与介绍

## 一、什么是字典表
字典表（也称为"维度表"或"映射表"）是数据管理和分析中常用的辅助性表格，用于存储某一类"代码-含义"或"ID-属性"映射关系。它帮助标准化主数据表中的字段，提升数据的可读性、可维护性和分析效率。

## 二、字典表的作用
1. **标准化数据**：将主表中的"代码"或"缩写"转换为易懂的描述。
2. **补充属性**：为主表字段补充更多属性（如单位、数据类型、分组等）。
3. **便于维护**：代码或分类变化时，只需维护字典表，无需批量修改主表。
4. **提升分析效率**：在Power BI等BI工具中，字典表可作为"维度表"与主表关联，便于切片、筛选和展示。

## 四、字典表的使用场景
- **数据清洗**：将主表中的"代码"字段通过查找字典表，转换为标准化的描述或补充属性。
- **数据建模**：在Power BI、Tableau等BI工具中，字典表作为"维度表"与主表（事实表）关联，实现多维分析。
- **数据校验**：通过字典表校验主表中的数据是否合法、标准。

## 五、字典表的维护建议
- 字典表应单独存放，便于集中管理和维护。
- 字典表内容应定期更新，确保与实际业务同步。
- 字典表字段应尽量全面，如代码、名称、单位、类型、说明等。
