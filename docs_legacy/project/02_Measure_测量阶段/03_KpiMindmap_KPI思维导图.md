## 指标关联关系
### 生产与质量
- 设备综合效率（OEE）影响一次合格率（FPY）
- 一次合格率（FPY）影响不合格品率（NCR）
- 不合格品率（NCR）影响CAPA完成率和返工率

### 安全相关
- 总可记录事故率（TRIR）影响安全事件率和可记录事故数量
- 安全事件率与隐患发现数量相关

### 成本相关
- 运营成本影响：
  - 成本节约
  - 项目节约
  - 采购成本节约
  - 维修成本
  - 设施维护成本

### 库存相关
- 库存周转率影响：
  - 总库存
  - 积压订单
